Core Features at a Glance

Everything you need for scroll-based presentations

Scroll Navigation

Smooth scroll-snap navigation with keyboard support. Works on any device.

Anchored Comments

Select text to add comments. Supports local storage or Power Automate backend.

PDF Export

Gated download unlocked on reaching the final slide. Uses Reveal.js print mode.

Canvas Animations

Per-slide animated canvases with play/pause on click. Great for diagrams.

Reactive State

Fine-grained reactivity with signals, memos, and effects. SolidJS-inspired.

No Build Required

Pure HTML, CSS, JS. Just open in a browser or serve statically.

Sample Content for Commenting

This paragraph is perfect for testing comments. Select any portion of this text with your mouse, and a comment dialog will appear. Your comments are anchored to the specific text you selected, making it easy to provide contextual feedback.

Comments support multiple reviewers. Each comment shows the author's name and timestamp. You can edit or delete your own comments. The draggable markers show where comments are anchored on the slide.

Key Features:

  • ✓ Text-anchored comments - Comments stay connected to specific text selections
  • ✓ Draggable markers - Reposition comment indicators anywhere on the slide
  • ✓ Offline support - Comments sync when connection is restored
  • ✓ Conflict detection - Handles concurrent edits gracefully
Tip: Click the "Comments" button in the top-right corner to toggle the comment panel and see all comments on this slide.
